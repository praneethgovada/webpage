Learning how to use basic linuc, vi and git commands

linux commands ls - directory listing cd - directory change pwd - change current directory mkdir - create a directory

vi commands i - insert Escape - get out of insert mode Shift key+colon - command prompt for vi W - command to write Q - command to read WQ - Write and Quit

git commands Git init - Create the repository Git Add filename - add the file(S) to git in preparation of a commit Git commit-m"message" - do the commit to save changes Git checkout-b = make a new branch and checkout Git merge - merge the branch note
